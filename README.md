# 📱 Mobile Sales Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Mobile Sales Dashboard built in Microsoft Power BI to analyze sales performance, customer purchasing behavior, product trends, and regional performance.

The dashboard transforms raw transactional data into meaningful business insights using Power Query for data preparation, DAX for KPI calculations, and interactive visualizations.

---

## 🎯 Business Problem

Businesses selling mobile phones often generate large volumes of sales data. Analyzing this data manually is time-consuming and makes it difficult to identify:

- Best-selling brands
- Revenue trends
- Customer purchasing behavior
- Regional performance
- Payment preferences
- Overall business performance

This dashboard provides decision-makers with an interactive solution to monitor sales and support data-driven decisions.

---

## 📂 Dataset

The dataset contains mobile phone sales transactions with the following information:

- Transaction ID
- Date (Day, Month, Year)
- Brand
- Mobile Model
- Units Sold
- Price Per Unit
- Customer Details
- Customer Rating
- City
- Payment Method

Total Records: **3,835**

## 📌 Key Business Insights

- Apple generated the highest overall revenue.
- Delhi recorded the highest sales among all cities.
- UPI was the most preferred payment method.
- Premium smartphones contributed significantly to total revenue.
- Customer ratings remained consistently positive across transactions.
